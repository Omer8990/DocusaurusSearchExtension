# Introduction

**Project B** is a modern, responsive UI component library for React. It focuses on usability and accessibility.

## Key Features

*   **Accessible:** WAI-ARIA compliant.
*   **Themable:** Easy to customize with CSS variables.
*   **Lightweight:** < 10kb gzipped.

## Quick Start

```bash
npm install @project-b/core
```
